# AutoFix Servis — Car Service Website

A modern single-page website for a fictional Czech car service **AutoFix Servis** based in Ostrava. Serves as a template/portfolio showcase.


## Tech Stack

- **HTML5** — semantic markup, accessibility (ARIA)
- **CSS3** — custom properties, grid, flexbox, animations, responsive design
- **Vanilla JavaScript** — Intersection Observer, smooth scroll, mobile menu
- **Google Fonts** — Inter

No frameworks, no dependencies.

## Features

- **Responsive design** — mobile-first, works on all devices
- **Dark theme** — with amber/orange accent (#f59e0b)
- **9 sections** — navigation, hero, services, pricing, about, gallery, reviews, contact, footer
- **Scroll animations** — fade-in effects using Intersection Observer
- **Active navigation** — highlights current section while scrolling
- **Mobile menu** — hamburger menu with smooth animation
- **Contact form** — with validation and success message (frontend only)
- **SVG icons** — inline, no external dependencies
- **Accessibility** — semantic HTML, ARIA attributes, keyboard navigation
- **SEO** — meta tags, Open Graph
